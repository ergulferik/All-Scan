# Proje Özeti: AllScan

## Proje Adı
AllScan - Web Sitelerinin Güvenlik Durumlarını Kullanıcılara Bildiren Chrome Eklentisi

## Proje Danışmanı
Doç. Dr. Bora Canbula

## Proje Sorumluları
Ergül Ferik, Ceyhun Binal

## Kurum
Manisa Celal Bayar Üniversitesi

## Özet
Kötü niyetli kişiler, kullanıcıların kişisel bilgilerini elde etmek için hileli bağlantılar kullanmaktadır. Diğer şeylerin yanı sıra, bu veriler hesap ele geçirme, finansal dolandırıcılık, kimlik hırsızlığı, kurumsal istilalar ve zararlı yazılımların yayılması için kullanılmaktadır.

Bu gelişmeler nedeniyle internet güvenliği her geçen gün daha da önemli hale geliyor. Yakın zamanda geliştirilen AllScan teknolojisinin amacı, kullanıcılara güvenli ve emniyetli bir internet deneyimi sunmaktır. Bu program, entegre antivirüs yazılımı sayesinde güvenlik kontrolleri yaparken bir web sayfasındaki her bağlantıyı sistematik olarak keşfetmek ve incelemek için tasarlanmıştır. Her bağlantının güvenliği gerçek zamanlı olarak güncellendiği için kullanıcılar internette güvenle gezinebilirler.

AllScan, algoritmalar kullanarak web sitelerinden alınan URL'leri güvenlik açıkları açısından titizlikle kontrol eder. Bağlantılar keşfedilir keşfedilmez, uygulama önde gelen antivirüs uygulamalarıyla birlikte taramaya başlar. Sonuçlar daha sonra kullanıcıya gösterilir ve hangi bağlantıların güvenli, hangilerinin tehlikeli olabileceğini belirtir.

AllScan'in kullanıcının tarayıcısıyla sorunsuz bir şekilde entegre olurken güvenliği artırma yeteneği, ana avantajlarından biridir. Arka planda çalışan uygulama sayesinde kullanıcılar çevrimiçi ortamda ne yapacakları konusunda bilinçli kararlar verebilirler. AllScan'in kullanıcı dostu arayüzü ve gerçek zamanlı ağ denetimi, internete erişimi daha güvenli hale getirir.

# Project Summary: AllScan

## Project Title
AllScan - Chrome Extension to Inform Users about the Security Status of Websites

## Project Advisor
Assoc. Prof. Bora Canbula

## Project Managers
Ergül Ferik, Ceyhun Binal

## Institution
Manisa Celal Bayar University

## Summary
Malicious individuals are using fraudulent links to get users' personal information. Among other things, these data are utilized for account takeovers, financial fraud, identity theft, corporate invasions, and the spread of harmful software.

Because of these advancements, internet security is becoming more and more crucial every day. The aim of the recently developed AllScan technology is to provide users with a safe and secure internet experience. This program is designed to systematically discover and inspect each link on a webpage while doing security checks, thanks to integrated antivirus software. Users may browse the internet with confidence since every connection's security is updated in real time.

Using algorithms, AllScan meticulously checks URLs taken from websites for security flaws. As soon as connections are discovered, the application begins scanning in tandem with leading antivirus applications. The results are then displayed to the user, indicating which connections are safe and which may be hazardous.

AllScan's ability to increase security while integrating seamlessly with the user's browser is one of its main advantages. With the application running in the background, users may make informed decisions about what they do online. AllScan's user-friendly interface and real-time network inspection make accessing the internet safer.
